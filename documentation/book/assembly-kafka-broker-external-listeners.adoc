= External listeners

An external listener allows a client outside {ProductPlatformName} to connect to a {ProductName} Kafka cluster inside {ProductPlatformName}.
{ProductName} supports the following types of external listeners:

* `route`
* `loadbalancer`
* `nodeport`
ifdef::Kubernetes[]
* `ingress`
endif::Kubernetes[]
