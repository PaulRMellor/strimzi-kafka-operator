Configuration points are examined, including options to secure and monitor Kafka.

A typical Kafka deployment is described, as well as the Operators used to deploy and manage Kafka.
Operators are fundamental to the running of {ProductName}.
The Operators provided with {ProductName} are purpose-built with specialist operational knowledge to effectively manage Kafka.
